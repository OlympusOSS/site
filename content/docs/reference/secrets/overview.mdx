Every secret material in an Olympus deployment, with its purpose and rotation path.

| Secret | Purpose | Critical |
|--------|---------|----------|
| [`ENCRYPTION_KEY`](./encryption-key) | Master key for SDK settings encryption. | **yes** |
| [`SESSION_SIGNING_KEY`](./session-signing-key) | HMAC for Athena session cookies. | **yes** |
| [`CIAM_RELOAD_API_KEY`](./ciam-reload-api-key) | Auth for CIAM Kratos schema reload sidecar. | — |
| [`IAM_RELOAD_API_KEY`](./iam-reload-api-key) | Auth for IAM Kratos schema reload sidecar. | — |
| [`CIAM_KRATOS_COOKIE_SECRET`](./kratos-cookie-secret-ciam) | Kratos session cookie HMAC. | — |
| [`IAM_KRATOS_COOKIE_SECRET`](./kratos-cookie-secret-iam) | Kratos IAM session cookie HMAC. | — |
| [`CIAM_KRATOS_CIPHER_SECRET`](./kratos-cipher-secret-ciam) | Kratos recovery/verification token cipher. | — |
| [`IAM_KRATOS_CIPHER_SECRET`](./kratos-cipher-secret-iam) | Kratos IAM cipher. | — |
| [`CIAM_HYDRA_SYSTEM_SECRET`](./hydra-system-secret-ciam) | Hydra encrypts client secrets and JWKs with this. | **yes** |
| [`IAM_HYDRA_SYSTEM_SECRET`](./hydra-system-secret-iam) | Hydra IAM system secret. | **yes** |
| [`SMTP_USER / SMTP_PASSWORD`](./smtp-credentials) | Email provider credentials. | — |
| [`TURNSTILE_SECRET_KEY`](./turnstile-secret-key) | Cloudflare Turnstile verification. | — |
| [`DATABASE_URL`](./database-url) | Postgres connection string with password. | — |
| [`Postgres CA certificate`](./postgres-ca-cert) | TLS verify-full root cert. | — |
| [`DEPLOY_SSH_KEY`](./ssh-deploy-key) | GitHub Actions deploy SSH key for the VPS. | — |
| [`GHCR pull token`](./ghcr-pull-token) | Pull private images from GHCR. | — |
| [`Daedalus provider tokens`](./daedalus-provider-tokens) | DigitalOcean / Hostinger / Neon API keys. | — |

See [Security — Secrets management](/docs/security/secrets-management) for the architectural overview.
