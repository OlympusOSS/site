---
title: Athena API
description: REST API exposed by the Athena admin dashboard
---

Athena exposes a REST API at `/api/*`. Every route is protected by a session cookie (see [Athena API Authentication](/docs/reference/api-athena-authentication)), only `/api/health` and `/api/auth/**` are public.
