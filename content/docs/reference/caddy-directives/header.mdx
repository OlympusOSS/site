# Caddy directive: `header`

**Purpose:** Set or remove HTTP headers on requests / responses.

## Olympus usage

Sets HSTS, X-Frame-Options, CSP base, Referrer-Policy. See [Security, Security Headers](/docs/security/infrastructure/security-headers).

See [Caddy directives reference](https://caddyserver.com/docs/caddyfile/directives/header).
