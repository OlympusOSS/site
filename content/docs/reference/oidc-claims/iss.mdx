# OIDC claim: `iss`

**Type:** `string`

**Source:** Hydra

## Description

Issuer URL. Should match your CIAM/IAM Hydra base URL.

## Where it appears

- ID tokens (always for standard claims).
- Userinfo endpoint response (with appropriate scope).
- Introspection response (for opaque tokens).
