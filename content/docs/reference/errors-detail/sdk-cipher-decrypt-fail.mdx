# `cipher_decrypt_fail`

**Source:** SDK

**HTTP:** 500

## Summary

Decryption of an encrypted setting failed.

## When this fires

Reading a setting whose ciphertext doesn't match the current ENCRYPTION_KEY.

## Common causes

- ENCRYPTION_KEY rotated without migration
- Ciphertext corruption

## Fix

See [Operate, Encryption key rotation](/docs/operate/rotation/encryption-key-rotation) or [Cookbook, Recover lost encryption key](/docs/cookbook/secrets-encryption/recover-lost-encryption-key).
