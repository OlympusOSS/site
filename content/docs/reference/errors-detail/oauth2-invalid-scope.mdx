# `invalid_scope`

**Source:** Hydra (RFC 6749)

**HTTP:** 400

## Summary

Requested scope is invalid or unknown.

## When this fires

Client requested a scope not in its allowed scope list.

## Fix

Add the scope to the client's allowed scopes in Athena.
