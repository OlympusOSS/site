Every known error code emitted by Olympus and its components, with what causes it and how to fix.

| Code | HTTP | Summary |
|------|------|---------|

| [`forbidden`](./athena-forbidden) | 403 | Session is valid but role is insufficient. |
| [`bad_content_type`](./athena-bad-content-type) | 415 | Mutation routes require `Content-Type: application/json`. |
