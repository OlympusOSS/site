# `encryption_key_blocklisted`

**Source:** SDK

**HTTP:** 500

## Summary

ENCRYPTION_KEY is on the blocklist of known-weak values.

## When this fires

Container startup, fatal.

## Fix

Generate a strong key: `openssl rand -base64 32`. See [Security, Encryption key blocklist](/docs/security/encryption-keys/encryption-key-blocklist).
