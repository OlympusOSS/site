# `unsupported_grant_type`

**Source:** Hydra (RFC 6749)

**HTTP:** 400

## Summary

Grant type not supported by the server.

## When this fires

Most often: trying to use `password` grant (removed in Olympus per OAuth 2.1).

## Fix

Use Authorization Code instead.
