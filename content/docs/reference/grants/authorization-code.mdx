# Authorization Code

**Spec:** RFC 6749 §4.1

**Supported in Olympus:** **Yes**

## When to use

Server-side web apps that can keep a secret.

## How it works

User redirects to Hydra → authenticates via Hera/Kratos → returns to app with one-time code → app exchanges code for tokens.
