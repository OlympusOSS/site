# Client Credentials

**Spec:** RFC 6749 §4.4

**Supported in Olympus:** **Yes**

## When to use

Backend-to-backend (M2M). No user involved.

## How it works

Confidential client POSTs `grant_type=client_credentials` with its credentials. Hydra returns an access token (no refresh, no ID token).
