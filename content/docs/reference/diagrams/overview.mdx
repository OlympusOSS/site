## How these render

Mermaid is built into Fumadocs via a custom remark plugin (see [site/lib/remark-mermaid.mjs](https://github.com/OlympusOSS/site/blob/main/lib/remark-mermaid.mjs) and the [Mermaid component](https://github.com/OlympusOSS/site/blob/main/src/components/site/mermaid.tsx)).

To add a new diagram: write the `sequenceDiagram` code in a `\`\`\`mermaid` fence in any MDX file. Same for `stateDiagram`, `graph TB`, etc.
