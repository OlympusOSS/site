# Port `4000` — Hera IAM

**Role:** Employee login UI

**Exposure:** intranet only

## Purpose

Internal employee login. Same constraints as Hera CIAM.

## Security

Internal-only. No host binding in prod.

## Verify exposure

```bash
# From outside the VPS
nmap -p 4000 <vps-ip>
# Expected: depends on your firewall config
```
