# Port `80` — Caddy

**Role:** HTTP ingress

**Exposure:** internet

## Purpose

Redirects to HTTPS (port 443).

## Security

Public exposure is expected, but defense-in-depth via Caddy.

## Verify exposure

```bash
# From outside the VPS
nmap -p 80 <vps-ip>
# Expected: open
```
