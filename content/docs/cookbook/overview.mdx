The Cookbook section is a growing set of recipes for things people actually want to do with Olympus: validate tokens in five different languages, use Olympus as the OIDC IdP for ArgoCD, add a new social IdP, rotate keys with zero downtime, and so on.

## Categories

- **Token validation** — Go, Node, Python, Rust.
- **OIDC integrations** — AWS, Grafana, ArgoCD, pgAdmin.
- **Social IdPs** — Google, Apple, Microsoft.
- **Customization** — registration form, email templates, consent screen, custom claims, scopes.
- **Operations** — key rotation procedures (zero-downtime), lost-key recovery scenarios, bulk identity import, GDPR DSR export.
- **MFA** — adding WebAuthn, enforcing step-up to AAL2.
- **Deployment** — single $5 VPS, managed Postgres, staging + prod on one host, cert-expiry alerts.
- **MCP** — driving deployments from Claude.

## Status

This section is built out in **Phase 8** of the documentation plan. ~35 recipes are pre-registered in `meta.json`; the full set will be filled out in priority order, driven by what real users ask for.
