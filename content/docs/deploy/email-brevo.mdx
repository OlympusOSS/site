[Brevo](https://brevo.com) (formerly Sendinblue) offers transactional email with a generous free tier (300 emails/day, no monthly cap).

## Configuration

```yaml
courier:
  smtp:
    connection_uri: smtp://<smtp_login>:<smtp_password>@smtp-relay.brevo.com:587/?skip_ssl_verify=false
    from_address: noreply@your-domain.com
    from_name: Your App
```

Brevo's SMTP credentials are separate from the API key. Get them from Brevo dashboard → SMTP & API → SMTP tab.

For Daedalus: choose **Brevo** in the Email wizard; paste both SMTP login and password.

## Domain authentication

Brevo requires DKIM verification before reliable delivery:

1. Brevo dashboard → Settings → Domain Authentication.
2. Add your sending domain.
3. Add the DKIM and DMARC DNS records.
4. Wait for verification.

## Free tier

- 300 emails/day (~9,000/month at consistent use).
- Both transactional and marketing in one account.

## Caveats

- Brevo branding by default ("Sent via Brevo" footer). Remove via paid plan or upgrade.
- The free plan has a daily cap, not a monthly one — handy for steady traffic, awkward for bursts.
