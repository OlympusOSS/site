Daedalus walks operators through 13 ordered steps. Each has its own page.

| # | Step | Purpose |
|---|------|---------|
| 1 | [Repository](./repository) | Fork OlympusOSS/platform and clone locally. |
| 2 | [Domain](./domain) | Configure apex + 3 subdomains (CIAM, IAM, Site). |
| 3 | [Provider](./provider) | Choose VPS provider. |
| 4 | [Email](./email) | Configure outbound email for verification/recovery. |
| 5 | [Database](./database) | Provision Postgres. |
| 6 | [Compute](./compute) | Create the VPS. |
| 7 | [Secrets](./secrets) | Generate cryptographic secrets. |
| 8 | [OAuth](./oauth) | Register OAuth2 clients in Hydra. |
| 9 | [Deploy](./deploy) | First production deploy. |
| 10 | [Health](./health) | Verify every service is responsive. |
| 11 | [Accounts](./accounts) | Seed the first IAM admin. |
| 12 | [Applications](./applications) | Register Athena, Hera, Site as managed apps. |
| 13 | [Destroy](./destroy) | Full teardown (one-way). |

See [Deploy, With Daedalus wizard](/docs/deploy/wizards/with-daedalus-wizard) for the consolidated overview.
