This section covers production deployments, the path from "Olympus runs on my laptop" to "Olympus serves real customers on real DNS."

## Two paths

- **[Deploy with Daedalus](/docs/deploy/wizards/with-daedalus-wizard)**, the guided thirteen-step wizard. Best for most operators. Handles VPS provisioning, DNS, TLS, GitHub Secrets, and the first deploy.
- **[Deploy without Daedalus](/docs/deploy/wizards/without-daedalus-manual)**, manual production deploy for operators who already have infrastructure-as-code or a non-supported provider.
