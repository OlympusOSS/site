Linode (acquired by Akamai) offers VPS plans starting at $5/mo. Use Daedalus's [Direct SSH](/docs/deploy/providers/provider-direct-ssh) path.

## Recommended plan

| Plan | RAM | vCPU | Storage | Monthly |
|---|---|---|---|---|
| Nanode 1GB | 1GB | 1 | 25GB | $5 |
| Linode 4GB | 4GB | 2 | 80GB | $24 |
| Dedicated 4GB | 4GB | 2 | 80GB | $36 |

For Olympus, recommend at least Linode 4GB (or Nanode 1GB only if using managed Postgres + low traffic).

## Setup

1. Create a Linode instance (Compute → Linodes → Create → Ubuntu 22.04).
2. SSH in, install Podman as in the [Vultr recipe](/docs/deploy/providers/provider-vultr).
3. Use Daedalus's Direct SSH provider.

## Object Storage for backups

Linode's S3-compatible Object Storage is ideal for Postgres backup off-host:

```bash
rclone copy /backups/ linode:olympus-backups/
```

Pricing: $0.02/GB/month.

## Akamai DDoS protection

Linode includes basic DDoS protection. For Olympus's login endpoints, complement with Caddy's `rate_limit` ([Security, Rate Limiting](/docs/security/web-attacks/rate-limiting)) and Cloudflare Turnstile ([Security, Captcha](/docs/security/web-attacks/captcha-turnstile)).
