Operating an identity platform is a different shape of work from building one. This section is the runbook collection, the things you do on a Tuesday afternoon when something breaks, when a key needs to rotate, when an account is locked.

## When you are paged

- [Incident Response](/docs/operate/backups-recovery/incident-response), the on-call playbook.
- [Health Endpoints](/docs/operate/monitoring/health-endpoints), what each service's `/health` does and doesn't tell you.
- [Network Topology](/docs/operate/administration/network-topology), which ports must be firewalled, and which exist for in-cluster traffic only.

## Scheduled rotations

- [Certificate rotation](/docs/operate/rotation/cert-rotation), Caddy + database TLS certs.
- [Session signing key rotation](/docs/operate/rotation/session-signing-key-rotation), zero-downtime.
- [Encryption key rotation](/docs/operate/rotation/encryption-key-rotation), for the SDK settings vault.
- [Reload API key rotation](/docs/operate/rotation/reload-api-key-rotation), for the Kratos schema reload sidecar.
- [Secrets audit](/docs/operate/monitoring/secrets-audit), quarterly inventory.

## User support

- [Locked account unlock](/docs/operate/backups-recovery/locked-account-unlock), diagnose and unlock a customer or employee.
- [pgAdmin DBA offboarding](/docs/operate/administration/pgadmin-dba-offboarding), remove a departing DBA's database access cleanly.

## Observability

- [Monitoring, PKCE analytics](/docs/operate/monitoring/monitoring-pkce-analytics), PKCE-protected OAuth2 traffic.
- [Monitoring, MFA stats](/docs/operate/monitoring/monitoring-mfa-stats), second-factor enrollment and challenge rates.
- [Logs and observability](/docs/operate/monitoring/logs-and-observability), what's logged, where it goes, retention.
