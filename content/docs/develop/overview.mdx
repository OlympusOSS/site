This section is for **licensees inspecting the containers they are running** and for engineers contributing fixes back upstream.

The Olympus source is provided under the [Olympus Free Container License](/docs/license) — you can inspect any container you have running, patch locally, and contribute fixes; you cannot redistribute modified source as a competing product.

- [Inspecting your containers](/docs/develop/inspecting-your-containers) — how to read the source of a running container under the OFCL.
- [Workspace setup](/docs/develop/workspace-setup) — clone the eight repos as siblings, install Bun, configure the source-only `@olympusoss/*` packages.
- [Package naming](/docs/develop/package-naming) — why the npm packages are source-only and how that works.
