This section gets Olympus running for the first time. Pick your path:

- **[Quickstart with octl](/docs/get-started/quickstart-octl)**, fastest. One command. For trying Olympus on your laptop.
- **[Quickstart with podman compose](/docs/get-started/quickstart-podman-compose)**, manual. For understanding what `octl deploy` actually does.
- **[Quickstart with Daedalus](/docs/get-started/quickstart-daedalus)**, if you already want to deploy to a real VPS, skip the local stack and use the wizard.

Then:

1. [Access URLs and credentials](/docs/get-started/access-urls-and-credentials), log into Athena, the admin dashboard.
2. [CIAM vs IAM](/docs/get-started/concepts-ciam-vs-iam), the most important concept in Olympus.
3. [Identity vs OAuth2](/docs/get-started/concepts-identity-vs-oauth2), why we run both Kratos and Hydra.
4. [Your first login](/docs/get-started/first-login), exercise the login flow end-to-end.
5. [Your first OAuth2 client](/docs/get-started/first-oauth-client), register a client in Hydra and complete an authorization-code flow.
6. [Next steps](/docs/get-started/next-steps), where to go from here.

Estimated time end-to-end: **15 minutes**.
