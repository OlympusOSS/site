# Welcome to Olympus

Olympus is a self-hosted identity platform that provides authentication and authorization for modern applications. Built on [Ory Kratos](https://www.ory.sh/kratos/) (identity management) and [Ory Hydra](https://www.ory.sh/hydra/) (OAuth2/OIDC), it gives you full control over your users' identity data with no vendor lock-in.

## Key Features

- **Dual-domain architecture** — separate CIAM (customer-facing) and IAM (employee-facing) domains with independent identity and OAuth2 services
- **Full user lifecycle** — registration, login, account recovery, email verification, and profile management
- **OAuth2 and OpenID Connect** — authorization code flow, client credentials, PKCE, and OIDC discovery
- **Admin dashboard** — manage identities, sessions, OAuth2 clients, and schemas through Athena
- **Self-hosted** — deploy anywhere with Podman Compose, your infrastructure, your rules
